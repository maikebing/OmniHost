Web Stack Examples

把 ASP.NET / Blazor / Vue / 第三方网站 都放进 OmniHost。

这个样例会先启动本地 ASP.NET Core 服务，再由 OmniHost 打开不同窗口。 这样你可以直接比较 MVC、Blazor、Vue3 SPA、外部网站，以及脚本化 API 的接入方式。

1. ASP.NET MVC Core

Controller + Razor View

适合服务端渲染页面较多、现有 MVC 代码可以直接复用的场景。

2. Blazor + ASP.NET Core API

前端组件交互 + 后端 JSON

Blazor 页面交互由组件负责，数据接口仍然走标准 ASP.NET Core。

3. ASP.NET Core SPA + Vue 3

静态 SPA + 本地 API

用 Vue 3 写前端，ASP.NET Core 负责静态文件和接口。

4. 第三方网站

直接打开外部地址

示例用微软官网，适合包裹一个已有站点做桌面入口。

6. C# 脚本 API

.csx 生成后端返回

这里用 Roslyn C# Script 作为脚本层，让接口逻辑写在单独的脚本文件里。
